# UI Sounds — Default Theme

Положи сюда .ogg файлы для стандартной темы UI:

- `hover.ogg`        — наведение курсора на элемент
- `click.ogg`        — клик по элементу
- `screen_open.ogg`  — открытие экрана
- `screen_close.ogg` — закрытие экрана
- `overlay_open.ogg` — открытие оверлея
- `overlay_close.ogg`— закрытие оверлея
- `error.ogg`        — ошибка / недоступное действие
- `success.ogg`      — успешное действие
- `notification.ogg` — уведомление
